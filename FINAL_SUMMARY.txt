🚀 快速部署（3步）

1️⃣ 克隆项目
   git clone <your-repo> asterdex-trading-bot
   cd asterdex-trading-bot

2️⃣ 一键安装
   bash deploy/install.sh

3️⃣ 配置运行
   cp config/config.example.json config/config.json
   vim config/config.json  # 填写钱包地址和私钥
   python src/main.py
